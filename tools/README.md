# Tools

Useful tools for use during the Plugfest.

## Pre-built CLI Tools
Two open-source tools are pre-built and packaged in this repo  only for ease of access during the Plugfest event. Please check out the source repositories for more information!

NIST OSCAL-CLI: https://github.com/usnistgov/oscal-cli

Metaschema Framework OSCAL-CLI Fork: https://github.com/metaschema-framework/oscal-cli

These tools provide easy access to core OSCAL functionality: 
1. Metaschema validation
2. Metaschema to JSON and XSD schema generation
3. OSCAL document validation
4. Profile Resolution
